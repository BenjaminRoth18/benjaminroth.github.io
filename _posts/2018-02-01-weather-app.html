Going native.

A few weeks ago I decided to develop my first native app with NativeScript and Angular. The idea was to create a simple app using a random service to fetch some data.

I was discovering some thoughts and ideas an finally checked the forecast.io API which provides all necessary data of weather information.

Features

This app includes the following features in version 1.0.

  • Search locations
  • Weather for your current location
  • Hourly and weekly forecast
  • Weather layouts with colors
  • Your last search results
  • RadSideDrawer integration
  • Application settings storage

API

  • Weather data: Forecast.io
  • Geocoding: Google Maps
